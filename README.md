# 🧠 智谱AI多图分析工具

一个简单易用的桌面应用，支持拖拽上传多张图片，批量调用智谱AI（ZhipuAI）的视觉模型进行分析，并自动保存结果。

![截图](assets/screenshot.png)

## ✨ 功能特性

- ✅ 拖拽上传多张图片（支持 JPG/PNG/BMP/WebP）
- ✅ 批量调用 `glm-4v` 或 `glm-4.5v` 视觉模型
- ✅ 自定义系统提示词与用户问题
- ✅ 分析结果实时显示 + 自动保存为文本
- ✅ 支持配置记忆（API Key、提示词等）
- ✅ 暗色主题，美观易用
